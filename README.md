# apiDropbox

This is short Tutorial of how to make a Dropbox viewer, which is a single page application that list out files and folders from our Dropbox account and allows us to show different folder contents bye updating the code.

- We are integrated with Vuex and be caching the contents of the dropbox folders. The Dropbox folder path should also be utilizing the store to make the app more efficient. We are also querying the API only when we need to.
